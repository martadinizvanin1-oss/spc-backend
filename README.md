# 🚗 Consulta Veicular

Site de consulta e verificação de documentos veiculares.

## O que é?

Plataforma para busca de informações sobre veículos, incluindo verificação de passagem por leilão e histórico documental.

## Tecnologias

- Node.js
- Vercel

## Como usar

1. Acesse o site
2. Digite a placa do veículo
3. Consulte o histórico
